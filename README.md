# Quant Research Lab

A research platform for analyzing and optimizing options trading strategies using statistical modeling and machine learning.# Quant Research Lab

A research-driven quantitative finance project focused on options pricing, volatility modeling, and systematic strategy evaluation.

## Objectives

* Build an options pricing engine from scratch using the Black-Scholes framework.
* Develop implied volatility and Greeks calculators.
* Research and backtest options-based trading strategies.
* Explore volatility forecasting techniques.
* Create a professional portfolio project demonstrating quantitative research and software engineering skills.

## Tech Stack

* Python
* NumPy
* SciPy
* PyTest
* Git
* VS Code
